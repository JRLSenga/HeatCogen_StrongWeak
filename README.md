# Heat Co-Generation: Strongly and Weakly Coupled
This serves as a repository for the paper Technoeconomic analysis of small modular reactors decarbonizing industrial process heat: the value of strongly-coupled cogeneration. The zipped file contains 2 folders: GenX_Software and Input Files, and an Instructions .docx file.

The Instructions.docx file provides the description on how to run the models.

# GenX_Software

This contains the software necessary to run and replicate the results.

# Input Files

This contains the input files used to get the results in the model.
